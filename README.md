## ⭐ Caso de Estudio Destacado

### 💳 Aplicación Web de Wallet

Este proyecto representa uno de los desafíos más importantes en mi proceso de aprendizaje en desarrollo web, 
ya que me permitió integrar múltiples conceptos técnicos en una solución funcional.

---

### Descripción.

Aplicación web desarrollada en Python que permite gestionar transacciones, visualizar saldo y administrar ingresos y gastos de usuarios.

---

### Desafío principal.

El mayor desafío fue implementar correctamente la lógica y la correcta sintaxis para que la aplicación funcionara de manera adecuada
considerando tanto las transacciones como la consistencia de los datos.

---

### Solución.

Se desarrolló una aplicación la cual cumplió con las expectativas, siendo adecuada para lo que se requería. Siendo persistente en la lógica de una
aplicación bancaria, así como la sintaxis adecuada para las transacciones.

---

### 🔧 Herramientas utilizadas

HTML5 / CSS / Bootstrap / JavaScript / jQuery

---

### Aprendizajes.

* Desarrollo backend con Python
* Manejo de lógica de negocio
* Gestión de datos

---

### Métricas de impacto.
